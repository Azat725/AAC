Пользовательские настройки цветов:
   
   В CSS можно определить пользовательские цвета с помощью ключевого слова `var`, чтобы 
   использовать их в различных частях стилей. Пример задания пользовательского цвета:

   
   :root {
       --main-color: #336699; /* Определяем пользовательский цвет */
   }

   h1 {
       color: var(--main-color); /* Используем пользовательский цвет для заголовков */
   }
   

2. Пользовательские настройки шрифтов:
   
   Также можно определить пользовательские шрифты, указав их семейство и размер. 
   Пример задания пользовательского шрифта:

   
   :root {
       --main-font: "Arial", sans-serif; /* Определяем пользовательский шрифт */
   }

   body {
       font-family: var(--main-font); /* Используем пользовательский шрифт для всего текста на странице */
   }
