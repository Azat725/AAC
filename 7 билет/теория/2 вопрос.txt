Поля (padding) и отступы (margin):
   
   - `padding`: Устанавливает отступ внутри элемента от его границ.
   - `margin`: Устанавливает отступ вокруг элемента между его границей и соседними элементами.

   Например, чтобы задать отступ слева и снизу для всех параграфов:

   
   p {
       padding-left: 10px; /* Отступ слева */
       padding-bottom: 20px; /* Отступ снизу */
   }
   

2. Границы (border) и контуры:
   
   - `border`: Устанавливает границу вокруг элемента.
   - `outline`: Устанавливает контур вокруг элемента, который не влияет на размер и положение элемента.
  
   Пример установки границы и контура для изображения и кнопки:

   
   img {
       border: 1px solid #000; /* Тонкая сплошная граница вокруг изображения */
   }

   button {
       border: 2px dashed #f00; /* Пунктирная граница красного цвета для кнопки */
       outline: 2px solid #00f; /* Сплошный синий контур вокруг кнопки */
   }
