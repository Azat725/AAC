1. Переход внутри одного документа:

   Для создания ссылки на якорь внутри того же документа используется тег `<a>` с 
   атрибутом `href="#название"`. Например:

   
   <a href="#ссылкаНаЯкорь">Перейти к якорю</a>
   <h2 id="ссылкаНаЯкорь">Это якорь</h2>
   

2. Переход к другому документу:

   Для перехода на другую страницу используется атрибут `href="URL_другой_страницы"`. Например:

   
   <a href="https://www.example.com">Перейти на другую страницу</a>
   

3. Переход к метке другого документа:

   Для перехода к метке в другом документе используется атрибут `href="URL_другой_страницы#метка"`. Например:

   
   <a href="https://www.example.com#метка">Перейти к метке на другой странице</a>
   

4. Имена файлов и ссылки на них:

   Для создания ссылки на файл используют тег `<a>` с атрибутом `href="путь_к_файлу"`. Например:

   
   <a href="документ.pdf">Ссылка на файл PDF</a>
