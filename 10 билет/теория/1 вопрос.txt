1. Усиление текста:
   
   Для усиления (жирного стиля) текста в HTML используется тег `<strong>`. Пример:

   
   <strong>Этот текст будет усилен</strong>
   

2. Изменение размера шрифта:
   
   Для изменения размера шрифта можно использовать атрибут `style="font-size: значение;"`. Например:

   
   <p style="font-size: 20px;">Этот текст будет иметь размер шрифта 20 пикселей</p>
   

3. Выравнивание абзацев и фрагментов:
   
   Для выравнивания текста по горизонтали существуют следующие атрибуты:
   - `text-align: left;` (по левому краю)
   - `text-align: right;` (по правому краю)
   - `text-align: center;` (по центру)
   - `text-align: justify;` (по ширине)

   Например, для выравнивания текста по центру:

   
   <p style="text-align: center;">Этот текст будет выровнен по центру</p>
   

4. Специальные символы:
   
   Чтобы использовать специальные символы, такие как &copy; (©), &nbsp; (неразрывный пробел) и др., 
   нужно использовать их HTML коды. Например:

   
   &copy; <!-- Выводит символ копирайта © -->
   &nbsp; <!-- Вставляет неразрывной пробел -->
