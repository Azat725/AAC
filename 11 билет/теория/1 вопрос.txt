HTML предоставляет различные теги для создания различных видов списков. Вот как можно создать:

1. Маркированный список:
<ul>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
  <li>Пункт 3
    <ul>
      <li>Подпункт 3.1</li>
      <li>Подпункт 3.2</li>
    </ul>
  </li>
</ul>


2. Нумерованный список:
<ol>
  <li>Пункт 1</li>
  <li>Пункт 2</li>
  <li>Пункт 3
    <ol>
      <li>Подпункт 3.1</li>
      <li>Подпункт 3.2</li>
    </ol>
  </li>
</ol>


Дополнительно, для контроля стилизации списка и отступов, вы можете использовать CSS. Например, чтобы создать круглые маркеры для маркированного списка:
ul {
  list-style-type: circle;
}


Или изменить нумерацию для нумерованного списка:
ol {
  list-style-type: roman;
}
