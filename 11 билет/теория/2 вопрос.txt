Для вставки картинки в HTML документ используется тег <img>. Вот пример кода:

<img src="путь_к_картинке.jpg" alt="Описание картинки">


- src - указывает путь к изображению
- alt - представляет альтернативный текст, который будет показан, если изображение не может быть загружено или для людей с ограниченными возможностями

Для выравнивания изображения и добавления рамки можно использовать атрибуты align и border:

<img src="путь_к_картинке.jpg" alt="Описание картинки" align="left" border="2">


- align может принимать значения: left, right, top, bottom
- border указывает ширину рамки в пикселях

Для создания изображения, которое будет являться ссылкой, вы можете обернуть тег <img> тегом <a>:

<a href="ссылка_на_страницу.html">
  <img src="путь_к_картинке.jpg" alt="Описание картинки">
</a>


Чтобы установить изображение в качестве фона всей страницы, вы можете использовать CSS:

body {
  background-image: url("путь_к_картинке.jpg");
  background-size: cover;
}
