Ширина и высота таблицы:

- Для задания ширины таблицы в CSS используется свойство width. Например, чтобы установить ширину 
таблицы в 100%, можно сделать следующее:
table {
  width: 100%;
}


- Для задания высоты таблицы необходимо учитывать, что в большинстве случаев высота таблицы 
автоматически регулируется контентом внутри ячеек. Однако, можно использовать свойство height для
установки фиксированной высоты таблицы:
table {
  height: 200px;
}


Вертикальное и горизонтальное выравнивание:

- Вертикальное выравнивание содержимого ячеек таблицы:
  - Для вертикального выравнивания содержимого ячеек таблицы по центру можно использовать свойство 
  vertical-align в CSS. Например:
  
  td {
    vertical-align: middle;
  }
  
  - Для вертикального выравнивания содержимого ячеек таблицы вверх можно использовать top:
  
  td {
    vertical-align: top;
  }
  

- Горизонтальное выравнивание содержимого ячеек таблицы:
  - Для горизонтального выравнивания содержимого ячеек таблицы по центру можно использовать свойство 
  text-align. Например:
  
  td {
    text-align: center;
  }
  
  - Для горизонтального выравнивания содержимого ячеек таблицы по левому краю можно использовать left:
  
  td {
    text-align: left;
  }
