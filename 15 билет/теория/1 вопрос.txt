Построение таблиц на веб-странице с помощью тега <table> в HTML очень важно для отображения 
табличных данных. Вот основные элементы и атрибуты, которые используются при создании таблиц:

1. Тег <table>:
   - Тег <table> используется для создания таблицы на веб-странице.
   - Атрибуты тега <table>:
     - border - определяет ширину границ таблицы.
     - width - определяет ширину таблицы.
     - align - выравнивание таблицы (left, right, center).

2. Теги <tr> и <td>:
   - Тег <tr> (table row) используется для создания строк в таблице.
   - Тег <td> (table data) используется для создания ячеек данных в строке.
   - В каждой строке (<tr>) может быть несколько ячеек данных (<td>).

3. Атрибуты тега <table>:
   - border - определяет толщину границ таблицы. Пример: <table border="1">.
   - width - определяет ширину таблицы. Может быть в процентах или пикселях. Пример: <table width="100%">.
   - cellpadding - определяет отступ внутри ячеек таблицы. Пример: <table cellpadding="5">.
   - cellspacing - определяет расстояние между ячейками таблицы. Пример: <table cellspacing="5">.

4. Вложенные таблицы:
   - Вложенные таблицы - это таблицы, которые находятся внутри другой таблицы.
   - Для создания вложенных таблиц вставьте тег <table> внутрь ячейки (<td>) основной таблицы.

Пример создания таблицы с вложенными таблицами:
<table border="1">
  <tr>
    <td>Ячейка 1</td>
    <td>Ячейка 2
        <table border="1">
            <tr>
                <td>Вложенная ячейка 1</td>
                <td>Вложенная ячейка 2</td>
            </tr>
        </table>
    </td>
  </tr>
  <tr>
    <td>Ячейка 3</td>
    <td>Ячейка 4</td>
  </tr>
</table>


Этот пример демонстрирует создание таблицы с двумя строками и двумя столбцами, где вторая 
ячейка второй строки содержит вложенную таблицу.
